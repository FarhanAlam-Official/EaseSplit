# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Initial project setup
- Core bill splitting functionality
- Group management system
- Expense tracking with multiple split options
- Analytics and visualization dashboard
- Export/import capabilities (PDF, CSV, JSON)
- Responsive design for all device sizes
- Dark mode support

### Changed

- Updated branding from "PayUno" to "EaseSplit"
- Updated email domain from "payuno.app" to "easesplit.app"
- Updated localStorage keys from "payuno_v1" to "easesplit_v1"

### Fixed

- Various UI/UX improvements
- Performance optimizations
- Bug fixes in expense calculations

## [0.1.0] - 2025-11-26

### Addeed

- Initial release of EaseSplit
- Privacy-first bill splitting application
- No-account-required usage
- Local storage only - no external servers
- Beautiful, modern UI with Tailwind CSS
- Responsive design for mobile and desktop
- Export to PDF, CSV, and JSON formats
- Smart settlement suggestions
- Expense categorization and analytics
